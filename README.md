# Описание
Игра в жанре булетхэл на языке Python с использованием модуля Arcade. Игрок, управляя космическим кораблём, должен уклоняться от летящих снарядов. Игра заканчивается если игрок получил урон 3 раза. Из времени жизни корабля формируется итоговый рекорд.

# Техническое задание
### Обзор интерфейса
Игра начинается с экрана главного меню. При нажатии кнопки "start" начинается геймплей. Игрок может посмотреть рекорды в виде таблицы после окончания игры, или через кнопку в меню.
### Обзор геймплея
Игрок, управляя космическим кораблём, должен уклоняться от летящих снарядов. Снаряды делятся на разрушаемые и неразрушемые. Игрок может стрелять в разрушаемые снаряды чтобы их уничтожить. Игра заканчивается если игрок получил урон 3 раза. Итоговый рекорд формируется из времени жизни корабля + доп. баллы за уничтоженные снарядов.
### Экраны
Игра содержит следующие экраны:
- __Меню__ - Главное меню игры. Содержит 3 кнопки: "start", "records", "exit".
- __Рекорды__ - Экран таблицы рекордов.
- __Игра__ - Содержит основной геймплей игры.
- __Пауза__ - Экран паузы.
### Основные классы
В коде игры присутствуют следующие классы:
- __MenuView__, __RecordsView__, __GameView__, __PauseView__ - Классы окон игры.
- __Ship__ - Космический корабль под управлением игрока.
- __Bullet__ - Пуля игрока.
- __ProjectileA__ - Неразрушаемый снаряд.
- __ProjectileB__ - Разрушаемый снаряд.

### Описание идеи
Игра на языке Python с использованием библиотеки Arcade. Цель игры - продержаться максимальное количество времени, уклоняясь от снарядов.
### Описание реализации
Игрок управляет кораблём при помощи клавиатуры и стреляет при помощи мышию. В игре присутствует функция "Рывка" - корабль ускоряется при зажатой кнопке. Ускорение тратит энергию, которая восстанавливается со временем. Снаряды появляются за экраном. Они имеют разные формы и скорости. Снаряды делятся на разрушаемые и неразрушаемые. Игрок может уничтожить ражрушаемые снаряды выстрелив в них.

Папка с игрой содержит 7 python файлов:
- __main__ - Пусковой файл для игры.
- __menu_view__ - Файл, содержащий класс окна главного меню.
- __game_view__ - Файл, содержащий класс окна игры.
- __pause_view__ - Файл, содержащий класс окна паузы.
- __score_view__ - Файл, содержащий класс окна таблицы рекордов.
- __player__ - Файл, содержащий классы игрока и пули.
- __projectiles__ - Файл, содержащий классы снарядов.

Помимо python файлов, папка игры содержит в себе папку со спрайтами.
### Описание технологий
Данный проект использует библиотеку Arcade и её основные классы, SQLite3 для работы с БД.
